Detailed Description

Driver to a VNH7100AS H-Bridge that can be connected to the VIA ARDUINO MEGA2560 Shield rev. 2.0.

The H-Bridge signals must be connected to J15 (Timer signals) on the VIA ARDUINO MEGA2560 Shield rev. 2.0. If current measuring is used, then H-Bridge signal CS must be connected to J12 (Analogue signals) on the VIA ARDUINO MEGA2560 Shield rev. 2.0.

Note
Current measuring is not implemented in this version of the driver!
