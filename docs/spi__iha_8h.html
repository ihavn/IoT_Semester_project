SPI Driver return codes.

These are the codes that can be returned from calls to the driver.

Enumerator
SPI_OK 

Everything OK

SPI_NO_ROOM_IN_TX_BUFFER 

Not enough room in TX buffer to store value, or buffer is not defined.

SPI_BUSY 

SPI bus is busy - nothing done.

SPI_ILLEGAL_INSTANCE 

The specified instance is > SPI_MAX_NO_OF_INSTANCES or is not instantiated yet.
